Start shopping:

Here You can start using main functionality of application. Application delivers two way of setting maximum amount of expenses:
> when user click "Add expense" - app uses default amount configured in application settings
> when user click "Enter max amount" - user can manually enter maximum amount for given shopping

Settings

Here user can set following default values:
> Balance - amount of value of wallet for given month
> Default currency - currency which is defalut choosen on add expenses screen
> The amount of one expense - maximum amount of one shopping

Expenses screen

This screen contain following elements:
> Moneys left - amount of moneys left in your wallet for given shopping
> Expense - here will appear entered price
> Choose currency - list with available currencies for given product
> Cathegory of expense - "Other" is selected as default
> Keyboard with numbers - using "," user can insert float prices and using "X" user can mulitple price
> ADD - adding entered price to list
> REMOVE - removes last entered sign
> FINISH - saves your expenses and returning to main menu
> CANCEL - returning to main menu without saving your expenses
> Show expenses - displays list of last added expenses
